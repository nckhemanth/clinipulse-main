# CliniPulse Web

The CliniPulse AI clinical workspace is built with Next.js, React and TypeScript. It provides medical-document upload and management, grounded clinical chat, analytics, email reporting and access-request workflows.

Set `NEXT_PUBLIC_API_URL` to the Spring Boot API URL before running or building the application.

```bash
npm install
npm run dev
```

The interface is designed for a controlled clinical environment. Authentication and authorization are enforced by the API and must remain enabled in deployed environments.
